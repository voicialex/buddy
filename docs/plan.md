# Buddy Robot 实施计划（ROS 2 现状版）

版本: v2.0  
日期: 2026-04-30  
状态: 执行中

## 1. 当前基线

已完成：

1. 组件化包结构与 `buddy_interfaces` 接口定义。
2. `buddy_app` 统一启动容器（C++ 入口）。
3. 各核心包具备基础 gtest。

## 2. 当前优先级

1. 完善状态机会话一致性（多句 TTS、打断恢复）。
2. 强化云端错误路径（重试、超时、日志可观测）。
3. 补齐跨包联调测试与回归基线。

## 3. 执行规则

1. 每次行为改动必须附带对应测试更新。
2. 新增参数统一进入 `buddy_app/params/*.yaml`。
3. 不引入新的历史兼容层（KISS/YAGNI）。
